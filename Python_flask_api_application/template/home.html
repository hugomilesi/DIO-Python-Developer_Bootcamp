Bem-vindo ao Projeto Simples de Aplicação Flask da DIO.

Visão Geral do Projeto

Neste projeto, o desafio final é criar e disponibilizar uma API desenvolvida utilizando o framework Flask, no ambiente colaborativo do Google Colab. O principal objetivo é realizar a leitura de dados de uma planilha no formato JSON por meio de uma API hospedada no Colab.

Detalhes

  • Criação de uma API com Flask.
  • Leitura de dados de uma planilha JSON.
  • Hospedagem e interação com a API no Google Colab.
